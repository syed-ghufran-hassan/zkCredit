# zkCredit


zkCredit is a privacy-preserving lending protocol on Mina Protocol that enables users to borrow and lend assets while keeping their financial history completely confidential. By leveraging zero-knowledge proofs, borrowers can verify their creditworthiness and collateralization without revealing sensitive wallet details or transaction data. The protocol empowers users to maintain full control over their financial privacy while accessing decentralized lending markets, aligning with Mina's vision of a lightweight, verifiable blockchain.
